# NG-Circos.js
NG-Circos - interactive circos for biological data

NG-Circos is an open source interactive Javascript library which provides an easy way to interactive display biological data on the web. It implements a raster-based SVG visualization using the open source Javascript framework D3.js. Circos.js is multiplatform and works in all major internet browsers (Internet Explorer, Mozilla Firefox, Google Chrome, Safari, Opera).
